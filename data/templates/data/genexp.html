GEO Data Analysis

Upload gene expression data from the Gene Expression Omnibus (GEO) data repository to obtain summary statistics and an estimate of transcription factor activity

What can I do with HTFome's GEO DataSet Analyser?

    HTFome's GEO DataSet Analyser allows you to:

    • Upload a GEO DataSet of your choice (in GDS format), containing gene expression data.
    • Obtain basic statistical information and publication-level graphical outputs, such as heatmaps, principal compenent analysis plots.
    • Visualise the inferred activity of your HTFs (obtained using DoRothEA and VIPER package algorithms).
    • Our analysis pipeline is fully interactive, allowing you to flexibly explore your gene expression data to the fullest.
    • Further features will be coming soon, such as volcano plots summarising differential gene expression analysis.

How can I access HTFome's GEO DataSet Analyser?

    • HTFome's GEO DataSet Analyser can be accessed through our GitHub repository.
    • All that is required is for you to download the app.R script (or you can copy/paste the code to your console) and run it from an IDE of your choice (preferably RStudio). Further details are provided on our GitHub.
    • The HTFome Dashboard (a Shiny dashboard app) should then appear where you can run your analysis.
    • We are currently working on deploying the HTFome Dashboard online, to allow for analysis in-browser.

Contact us if you experience any issues

If you experience any issues, or have any feedback, comments or even further features you would like implemented, please do get in touch with us by opening an issue on our GitHub.
